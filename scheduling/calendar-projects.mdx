## Calendar

**Support → Calendar** (or Events) shows company events on a calendar view.

### Events

Create events for:

- Maintenance windows
- Meetings
- Site visits
- Company holidays affecting support

Events may appear on dashboards and optional client communications.

### Calendar vs plan install

| Tool | Best for |
|------|----------|
| **Plan install** | Customer-specific install appointments |
| **Events** | General scheduling and maintenance |
| **Projects** | Multi-step work over days/weeks |

## Projects

**Support → Projects** tracks installation and engineering projects.

### Project views

- Active vs completed groupings
- Assignees, dates, status
- Link to clients or sites where configured

Use projects for:

- Fiber build segments
- Tower upgrades
- MDU deployments
- Internal IT initiatives

## Tickets and projects

Link tickets to projects when support work rolls into a larger initiative—or keep standalone for one-off fixes.

## Metrics

Dashboard **projects** and **events** cards summarize open counts; drill into lists for detail.

<Note>
Permissions may limit which staff can create or close projects—check with your admin if actions are missing.
</Note>
