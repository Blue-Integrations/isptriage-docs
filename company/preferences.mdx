**Company → Preferences** is the admin hub for company-wide settings. Tabs are linked via hash URLs (e.g. `#voip`, `#qoe`).

## General

- Company name and contact info
- Timezone
- Logo and background images (portal branding)
- Internal links (quick URLs for staff)
- Payment schedule and billing-related toggles where shown

## VoIP

- Enable **FSPBX** and/or **Thirdlane**
- Add server connections: name, URL, API token, verify SSL
- Multiple servers supported per provider

See [Email, DNS & VoIP](/communications/email-dns-voip).

## QoE (LibreQoS)

Configure **LibreQoS endpoints** for quality-of-experience monitoring:

- API URL and license key
- SSH access for advanced operations where used
- Test connection from endpoint list
- QoE data feeds AI assistant context and subscriber troubleshooting

<Note>
QoE endpoints moved from the Network nav into Preferences so all company integrations live in one place.
</Note>

## Import

CRM and billing system connections—see [CRM import](/company/crm-import).

## Email settings

Inbound ticket email accounts (IMAP), outbound SMTP identity for replies, and related ticket ingest options.

## Users

Staff user management may link from Preferences or **Company → Users**:

- Create staff with role
- Disable accounts
- Reset access

## Saving changes

Each tab has its own save action. After uploading images or tokens, wait for success flash before leaving the page.

<Warning>
Only **admin** users should access Preferences. Changes affect all staff and customers for the company.
</Warning>
